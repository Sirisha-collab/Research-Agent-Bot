# Research-Agent-Bot

Developed a bot with a React frontend and a Python (FastAPI/Flask) backend that enables users to upload research papers, retrieve relevant information, generate summaries, and automatically create research questions.

## Core Features
 Upload research papers (PDF).
 Retrieve relevant information using a RAG pipeline.
 Generate concise summaries.
 Automatically generate research questions from the summary.
 Support a multi-agent workflow for retrieval, summarization, and question generation.

## Tech Stack

### Frontend
React.js
Vite
Tailwind CSS (or Material UI)
Axios
React Router

### Backend
FastAPI (recommended) or Flask
Python
FAISS 
Gemini models
